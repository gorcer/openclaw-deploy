# Анализы для первичного скрининга

## БЛОК 1. ОБЩЕЕ САМОЧУВСТВИЕ
Если ≥2 пунктов → рекомендовать базовый скрининг
- постоянная усталость
- снижение энергии
- сонливость / бессонница
- раздражительность
- снижение концентрации

## БЛОК 2. НЕРВНАЯ СИСТЕМА
**Симптомы:** головные боли, тревожность, плохой сон, ухудшение памяти, внутреннее напряжение
**Анализы:** Витамин B12, Витамин D, Магний, ТТГ

## БЛОК 3. СЕРДЦЕ И СОСУДЫ
**Симптомы:** скачки давления, учащённое сердцебиение, одышка, холодные конечности, отёки
**Анализы:** Липидограмма, Глюкоза, HbA1c, Калий/натрий, ТТГ

## БЛОК 4. ПИЩЕВАРЕНИЕ
**Симптомы:** вздутие, изжога, запор/диарея, тяжесть после еды, горечь во рту
**Анализы:** АЛТ, АСТ, Билирубин, ГГТ, Helicobacter pylori, Кал на паразитов

## БЛОК 5. ЭНДОКРИННАЯ СИСТЕМА
**Симптомы:** колебания веса, повышенная потливость, чувствительность к холоду/жаре, упадок энергии, перепады настроения
**Анализы:** ТТГ, Т4 свободный, Глюкоза, Инсулин, HbA1c

## БЛОК 6. ОПОРНО-ДВИГАТЕЛЬНАЯ СИСТЕМА
**Симптомы:** боли в суставах, боли в спине, скованность, хруст, мышечная слабость
**Анализы:** Витамин D, Кальций, Магний, СРБ, Мочевая кислота

## БЛОК 7. ИММУНИТЕТ
**Симптомы:** частые простуды, слабость, долгое восстановление, воспаления
**Анализы:** ОАК, СРБ, Витамин D

## БЛОК 8. КОЖА / ВОЛОСЫ / НОГТИ
**Симптомы:** выпадение волос, сухость кожи, высыпания, ломкость ногтей
**Анализы:** Ферритин, Витамин D, Цинк, ТТГ

## БЛОК 9. МОЧЕВЫДЕЛИТЕЛЬНАЯ СИСТЕМА
**Симптомы:** частое мочеиспускание, отёки, дискомфорт
**Анализы:** ОАМ, Креатинин, Мочевина

## БЛОК 10. СЕНСОРНЫЕ СИСТЕМЫ (Зрение/слух)
**Симптомы:** снижение зрения, шум в ушах, головокружение
**Анализы:** Глюкоза, HbA1c, ТТГ, СРБ

## БЛОК 11. РЕПРОДУКТИВНАЯ СИСТЕМА

### Для женщин:
**Симптомы:** сбои цикла, ПМС, снижение либидо, эмоциональные качели
**Анализы:** ФСГ, Эстрадиол, Пролактин, ТТГ

### Для мужчин:
**Симптомы:** снижение либидо, снижение энергии, проблемы с эрекцией, сложности с зачатием
**Анализы:** Тестостерон общий, ГСПГ, Пролактин, ТТГ (при необходимости: ЛГ, ФСГ, эстрадиол)
